# Libraro

This is a personal bookshelf content management system based on [flask](https://palletsprojects.com/p/flask/) and [isbnlib](https://pypi.org/project/isbnlib/).

**Libraro** is Esperanto for a collection of books. The root *lib* means book, and the suffix *ar* means a collection.  
*Biblioteko* is a librairy or a book-shelf, but **libraro** sounded cooler.
